## Windows Server Setup

### Step 1: Windows Server Installation
- Installed Windows Server on a virtual machine
- Assigned a static IP address

### Step 2: Active Directory Installation
- Installed Active Directory Domain Services (AD DS) role
- Installed DNS Server role

### Step 3: Domain Controller Promotion
- Created a new forest
- Domain name used: cyberlab.local

### Step 4: Verification
- Active Directory Users and Computers opened successfully
- Domain services running correctly
