## Architecture Overview

This virtual lab represents a small enterprise Windows environment.

### Components
- 1 Windows Server (Domain Controller)
- 1 Windows 10 Client Machine

### Network Details
- Domain Name: cyberlab.local
- Server IP Address: 192.168.1.10
- Client IP Address: 192.168.1.20
- Network Type: Internal / Host-only Network

### Description
The Windows Server acts as a Domain Controller providing centralized
authentication, authorization, and policy enforcement.
The client machine is joined to the domain to simulate real user activity.
